## 📋 To Complete Setup (You Need To Do These)

### 1. Create Supabase Account (5 min)
- Go to https://supabase.com
- Sign up (free)
- Create new project

### 2. Get Your Credentials (2 min)
- Go to Settings > API
- Copy Project URL
- Copy anon public key

### 3. Create .env File (1 min)
- Create `.env` in project root
- Add your URL and key

### 4. Create Database Tables (2 min)
- Follow the SQL in guides
- Enable real-time

### 5. Run Dev Server (1 min)
```bash
npm install
npm run dev
```

### 6. Test Real-Time Sync (2 min)
- Open two browsers
- Toggle a dish
- Watch it sync! ✨
